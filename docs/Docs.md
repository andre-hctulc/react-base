# Docs

## Conventions

-   `Slot Props` are capitalized

```ts
interface Props {
    Pre?: React.ReactNode;
    HeaderProps?: HeaderProps;
}
```

-   `Variants` should be snake cas. E.g. _full_screen_

## Storybook

We use [Storybook](https://storybook.js.org/docs) for **documentation** and **testing**!
